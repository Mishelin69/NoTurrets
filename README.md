# NoTurrets

**NoTurrets** removes all the turrets from the game (ideally).

# Requirements for mod to build

BepInExPack mod and LobbyCompatibility mod.

# Requirements to play

If host has this mod on everyone else has to have it to prevent chaos.

# Build

 ```bash
   ./build.sh
   ```
